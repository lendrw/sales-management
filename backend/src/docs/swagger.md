## Documentation with Swagger

[swagger-ui-express](https://www.npmjs.com/package/swagger-ui-express) is middleware for Express applications that lets you integrate the Swagger UI interface with your API, making it easier to view and interact with the API documentation generated by Swagger directly in the browser.

How it works:

- Documentation generation: Uses the [swagger-jsdoc](https://github.com/Surnet/swagger-jsdoc) library to generate documentation from JSDoc comments in your code.
- Serving the Swagger UI interface: Uses swagger-ui-express to serve the Swagger UI graphical interface at a specified endpoint (for example, /api-docs).

Main features:

- API documentation visualization: Allows developers to easily view and understand the API's different routes, methods, parameters, and responses.
- Interactive testing: Through the interface, it is possible to send requests to the API directly from the browser, testing endpoints without an external tool such as Postman.
- Dynamic updates: Any change to the API documentation (JSDoc comments, for example) is automatically reflected in the Swagger UI interface.

Installation:

```shell
npm install swagger-ui-express swagger-jsdoc

npm install -D @types/swagger-ui-express @types/swagger-jsdoc
```
